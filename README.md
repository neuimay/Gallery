# Gallery

A personal photo gallery project built with React and modern web technologies.

This project is currently under active development and continuously evolving.

---

## Overview

Gallery is designed as a lightweight and scalable image management system.

The goal of this project is to explore:

- Modern frontend architecture
- Cloud-based object storage
- Image processing workflow
- Clean data structure design
- Performance-oriented rendering

---

## Tech Stack

**Frontend**
- React
- Vite
- TypeScript

**Image Processing**
- Sharp

**Storage**
- Cloudflare R2

---

## ✨ Features
- ⚡ Built with React + Vite
- ☁️ Cloudflare R2 object storage
- 🖼 Automatic thumbnail generation (Sharp)
- 📂 Image base / meta / exif data separation
- 🌙 Light / Dark theme toggle
- 📱 Responsive layout
- 🚀 Production-ready build structure
- 🔄 R2 auto-sync ready architecture

---

## Project Structure
```
gallery/

│   ├── lib/          # Data loading logic
```
